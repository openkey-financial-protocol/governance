## 📖 About / Acerca de

Este repositorio contiene la **Constitución de Gobernanza**, los principios operativos y la documentación estratégica de **OpenKey Network**.

OpenKey es una infraestructura financiera descentralizada diseñada para operar en entornos de alta restricción. Utiliza una arquitectura celular donde el **Núcleo Criptográfico** es público y auditable, mientras que la **Infraestructura Operativa** ("Plomería") se mantiene privada para garantizar la resiliencia y la seguridad contra ataques dirigidos.

## 🛡️ Core Philosophy / Filosofía Central

### Open Core, Private Plumbing
- **Open Core:** La lógica criptográfica, los contratos inteligentes y los algoritmos de intercambio son de código abierto (MIT License) para garantizar la transparencia y la confianza.
- **Private Plumbing:** La configuración de servidores, las rutas de proxy, las claves de API y los scripts de despliegue son privados. Esta capa oculta es nuestra principal defensa contra la censura y los ataques de denegación de servicio.

### Cellular Architecture
La infraestructura se distribuye en múltiples "órganos" independientes (nodos de cómputo, almacenamiento, mensajería) para eliminar puntos únicos de fallo y permitir la operación autónoma incluso si componentes individuales son bloqueados.
