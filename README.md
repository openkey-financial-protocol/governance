## 📖 About

This repository contains the **Governance Constitution**, operational principles, and strategic documentation for **OpenKey Network**.

OpenKey is a decentralized financial infrastructure designed to operate in high-restriction environments. It employs a **Cellular Architecture** where the **Cryptographic Core** is public and auditable, while the **Operational Infrastructure** ("Plumbing") remains private to ensure resilience against targeted attacks and censorship.

## 🛡️ Core Philosophy

### Open Core, Private Plumbing
- **Open Core:** The cryptographic engine, OpenFang logic, and smart contracts are public (MIT License) to ensure transparency and trust.
- **Private Plumbing:** Server configurations, proxy routes, API keys, and deployment scripts are kept private. This hidden layer is our primary defense against censorship and denial-of-service attacks.

### Cellular Architecture
Infrastructure is distributed across multiple independent "organs" (compute nodes, storage, messaging) to eliminate single points of failure and allow autonomous operation even if individual components are blocked.
